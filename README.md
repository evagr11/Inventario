# 🚀 Sobre el Proyecto

**Inventario IES Lacimurga** es una aplicación web de inventario sencilla y funcional que permite gestionar el material deportivo y educativo de un centro. El sistema está diseñado para ser intuitivo, rápido y accesible, sin necesidad de bases de datos o backend complejos, funcionando directamente en el navegador.

### ✨ Características principales:

*   **🗂️ Navegación por Categorías:** Filtra el inventario por categorías principales (Balones, Ropa, Juegos de mesa) y subcategorías para encontrar rápidamente lo que buscas.
*   **🔍 Búsqueda Dinámica:** Un buscador inteligente que permite encontrar materiales por nombre, código o descripción, resaltando las coincidencias en tiempo real.
*   **👕 Gestión de Stock por Tallas:** Soporte específico para prendas de ropa, permitiendo registrar cantidades, ubicaciones y notas para cada talla disponible.
*   **📊 Vista Detallada:** Accede a una ficha completa de cada artículo con su imagen, descripción, stock total, ubicación y notas asociadas.
*   **📄 Exportación a Excel:** Genera un archivo `.xlsx` con el inventario filtrado actualmente en pantalla, facilitando la creación de informes y su uso fuera de la aplicación.
*   **📱 Diseño Responsive:** La interfaz se adapta a cualquier dispositivo, ya sea un ordenador de sobremesa, una tablet o un móvil.

# 🧠 Contexto del Proyecto

Este proyecto fue desarrollado de manera **totalmente desinteresada y altruista** para el IES Lacimurga, mi antiguo instituto. El objetivo era proporcionar una herramienta moderna y fácil de usar para reemplazar sistemas de inventario obsoletos o manuales, ayudando al personal del centro a gestionar su material de forma más eficiente. Actualmente, la aplicación está desplegada con datos de muestra, a la espera de que el instituto incorpore su inventario real.

# 📂 Estructura del Proyecto

La estructura del proyecto es minimalista, separando la lógica, los datos, los estilos y el contenido principal.

```bash
.
├── app.js             # Lógica principal de la aplicación (búsqueda, filtrado, renderizado)
├── data.js            # Base de datos estática con los materiales y categorías
├── index.html         # Estructura principal de la web
├── styles.css         # Estilos visuales de la interfaz
├── logo.png           # Logo del instituto
├── textlogo.png       # Logo con texto
└── images/            # Carpeta con las imágenes de los productos
    ├── ajedrez.png
    ├── balondefutbol.png
    └── ...
```

# 🚀 Cómo Utilizar el Proyecto

Al ser una aplicación web estática, no requiere instalación ni un servidor complejo.

### 1. Visualización

Simplemente abre el archivo `index.html` en tu navegador web preferido (Chrome, Firefox, etc.).

Alternativamente, puedes clonar el repositorio y servirlo localmente:

```bash
# 1. Clona el repositorio
git clone https://github.com/evagr11/Inventario.git

# 2. Navega a la carpeta del proyecto
cd Inventario

# 3. Abre el archivo index.html en tu navegador
# (en Windows/macOS, puedes hacer doble clic sobre el archivo)
```

### 2. Modificar el Inventario

Toda la información del inventario se encuentra en el archivo `data.js`. Para añadir, modificar o eliminar materiales, solo necesitas editar este fichero siguiendo la estructura definida.

*   **`CATEGORIAS`**: Define las categorías y subcategorías que aparecen en el menú lateral.
*   **`MATERIALES`**: Es un array de objetos donde cada objeto representa un artículo del inventario.

```javascript
// Ejemplo de edición en data.js

const MATERIALES = [
  // ...otros materiales
  {
    id: "nuevo-material",
    nombre: "Nuevo Material de Prueba",
    categoria: "balones", // Debe coincidir con un `id` de CATEGORIAS
    imagen: 'images/balondevoleibol.png',
    descripcion: "Descripción del nuevo material.",
    cantidad: 10,
    estanteria: "Est. Z — Estante 99",
    notas_item: "Material recién añadido."
  },
];
```

Una vez guardados los cambios en `data.js`, simplemente **recarga la página** en el navegador para ver el inventario actualizado.
